# Dental Clinic Website

## Project Overview

Dental Clinic is a basic multi-page website created using only HTML. This project was developed to practice HTML fundamentals and understand how different HTML elements are used to build a website.

## Pages Included

- Home
- About
- Services
- Doctors
- Clinic Timings
- Appointment
- Contact
- Thank You

## HTML Concepts Used

- Basic HTML page structure
- Heading tags
- Paragraph tag
- Line break (`br`)
- Horizontal rule (`hr`)
- Anchor tag
- Target attribute (`target="_blank"`)
- Clickable image
- Image tag
- Bold, Italic and Underline tags
- Big and Small tags
- Subscript and Superscript tags
- Pre tag
- Main tag
- Section tag
- Article tag
- Aside tag
- Div tag
- Span tag
- Ordered List
- Unordered List
- Tables
  - `table`
  - `tr`
  - `th`
  - `td`
  - `caption`
  - `thead`
  - `tbody`
  - `colspan`
- Forms
  - `form`
  - `action`
  - `label`
  - `input`
  - `radio`
  - `checkbox`
  - `textarea`
  - `select`
  - `submit`
- Class and ID attributes
- Iframe
