# HTML
HTML learning and practice


It includes dental clinic website project folder. Dental Clinic is a basic multi-page website created using only HTML. This project was developed to practice HTML fundamentals and understand how different HTML elements are used to build a website.

Structure of project folder is :

```text
dental-website/
│
├── index.html
├── about.html
├── services.html
├── doctors.html
├── timings.html
├── appointment.html
├── thankyou.html
├── contact.html
├── images/
├── videos/
└── README.md
```
